# Summit Proposal Pack

This folder contains a clean, submission-ready package for Databricks Summit.

## Files
- `SUMMIT_PROPOSAL.md` - full proposal narrative
- `CFP_FORM_ANSWERS.md` - copy/paste answers by length
- `SPEAKER_BIO_TEMPLATE.md` - speaker bio templates
- `SUBMISSION_CHECKLIST.md` - final quality and submission checks

## Recommended Submission Order
1. Finalize your personal details in `SPEAKER_BIO_TEMPLATE.md`.
2. Choose the abstract length required by the CFP form from `CFP_FORM_ANSWERS.md`.
3. Paste the final text and run through `SUBMISSION_CHECKLIST.md`.
